Detall dels tags - submit

Defineix un botó de finalització d'un formulari.

Estén el tag layout:submit ( http://struts.application-servers.com/doc/index.html#submit ) de Struts-Layout.

Atributs heredats

Els atributs heredats de layout:submit son: accesskey, alt, altKey, bundle, mode, onblur, onchange, onclick, ondblclick, onfocus, onkeydown, onkeypress, onkeyup, onmousedown, onmousemove, onmouseout, onmouseover, onmouseup, policy, property, reqCode, selectCode, selectName, selectProperty, style styleClass, tabindex, title, titleKey, valign, value.

Atributs específics de Canigó

Atribut Funció
key Clau del literal del texte a mostrar dins el botó
services
styleId identificador del tag, per lligar-lo amb la configuració injectada amb Springidentificador del tag, per lligar-lo amb la configuració injectada amb Spring

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:submit

Pàgina JSP

<fwk:form action="provesTagsSimple.do"
          styleId="myActionForm"
          reqCode="show"
          method="post"
          layout="true">
    (...)
    <fwk:submit key="jsp.provesTags.ProvesTagsSimple.boto1" styleId="boto1"/>
    (...)
</fwk:form

Arxius d'internacionalització

jsp.provesTags.ProvesTagsSimple.boto1=Primer botó
