Detall dels tags - grid

Defineix una quadrícula per distribuir els tags que contingui en columnes i files. Es composa de tantes columnes com es defineixin i les files que calguin per encabir-hi els tags que contingui.

Estén el tag layout:grid ( http://struts.application-servers.com/doc/tags/layout.html#grid ) de Struts-Layout

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/grid.html

Atributs heredats

Els atributs heredats de layout:grid son: cols, height, id, space, styleClass, width.

Atributs específics de Canigó

Atribut Funció
align  
borderSpacing inserta el número indicat de pixels en la vorera de cada cel.la, creant així una separació entre elles.

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:grid

L'exemple crea un grid dues columnes. Com que inclou sis tags, es crearan automàticament 3 files.

Pàgina JSP

<%@ include file="/WEB-INF/jsp/includes/fwkTagLibs.jsp" %>

<fwk:form action="provesTagsLayout.do"
          styleId="myActionForm"
          key="jsp.provesTags.ProvesTagsSimple.form"
          reqCode="show"
          method="post"
          layout="true">
    <fwk:grid cols="2">
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp1"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp2"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>

        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp3"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="8"
                  size="8"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp4"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>

        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp5"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp6"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="12"
                  size="12"
                  property="camp1"/>
    </fwk:grid>
</fwk:form>

Arxius d'internacionalització

Els elements de layout no son visibles i, per tant, no tenen textes asssociats.
