Detall dels tags - Google Maps

Aquests tags integren l'api de Google Maps v2 amb Canigó.

Per a utilitzar els mapes de google s'han creat dos tags:

  1. fwk:gMap: crea un mapa on es poden indicar diferents opcions.
  2. fwk:gMapMarker: afegeix marcadors sobre el mapa podent indicar events sobre cadascun d'ells.

Aquests tags no implementen totes les funcionalitats de Google Maps, podeu trobar més informació de l'api de Google a ( http://code.google.com/intl/es-ES/apis/maps/ ).

Per a poder utilitzar Google Maps és necessari sol- licitar un apiKey http://code.google.com/intl/es-ES/apis/maps/signup.html

Configuració Spring

Validar que existeix el tag configurat a canigo-services-web.xml i sinó afegir-lo.

<bean id="googleMapsTag" class="net.gencat.ctti.canigo.services.web.taglib.google.GoogleMapsTag">
    <property name="logService" ref="loggingService"/>
    <property name="i18nService" ref="i18nService"/>
</bean>

Afegir a l'action-servlet-* la configuració del tag indicant l'apiKey.
<beans>
  <bean name="/provesTagsSimple" class="net.gencat.ctti.canigo.provesTags.struts.action.ProvesTagsSimpleAction">
    <property name="tagsConfiguration">
      <map>
        <entry key="*">
          <list>
            <bean parent="googleMapsTag">

  </bean>
</beans>

Configuració js

Per a que el component funcioni correctament és necessari tenir el fitxer maps.js porporcionat en la part estàtica del framework colocada a scripts/google/maps.js


fwk:gMap

Mostra un mapa que s'ajusta en alçada i amplada al 100% del contenidor.

Atributs de visualització

Atribut Obligatori Funció
id Sí Identificador del mapa.
longitude No longitud corresponent amb una ubicació de la superfície terrestre. Valor entre -180 i 180 amb una precisió de 6 decimals.
latitude No latitud corresponent amb una ubicació de la superfície terrestre. Valor entre -90 i 90 amb una precisió de 6 decimals.
address No Si no coneixes la longitud i latitud, pots indicar l'adreça de la següent manera: carrer nº, ciutat, país
zoom No Valor numèric per indicar l'ampliació del mapa. Per defecte 12.

Atributs de Control

Atribut Obligatori Funció
showScaleControl No true / false Mostra o oculta l'escala del mapa. Defecte true
showMapTypeControl No true / false Mostra o oculta els tipus de mapes. Defecte true
showLargeMapControl No true / false Mostra o oculta el zoom i control del mapa. Defecte true
mapType No Tipus de mapa a mostrar:
- G_NORMAL_MAP
- G_SATELLITE_MAP
- G_HYBRID_MAP
Valor per defecte G_NORMAL_MAP

Atributs d'events

Atribut Obligatori Funció
enableDragging No true / false Permet moure el mapa. Per defecte true.
enableDoubleClickZoom No true / false Realitza un zoom al fer doble click. Per defecte false.
onclick No S'ha d'especificar el nom de la funció javascript que es cridarà al fer click sobre el mapa. Aquesta funció pot rebre 2 paràmetres latitud i longitud del lloc on s'ha realitzat el click.
onmousemove No S'ha d'especificar el nom de la funció javascript que es cridarà al moure el ratolí sobre el mapa. Aquesta funció pot rebre 2 paràmetres latitud i longitud del lloc on està el mouse.
onmouseout No S'ha d'especificar el nom de la funció javascript que es cridarà al sortir del mapa. Aquesta funció pot rebre 2 paràmetres latitud i longitud del lloc on ha sortit el ratolí.
onmouseover No S'ha d'especificar el nom de la funció javascript que es cridarà al entrar en el mapa. Aquesta funció pot rebre 2 paràmetres latitud i longitud del lloc on ha entrat el ratolí.

fwk:gMapMarker

Permet afegir marcadors al mapa.

Atributs

Atribut Obligatori Funció
longitude No longitud corresponent amb una ubicació de la superfície terrestre en la que es mostrarà el marcador. Valor entre -180 i 180 amb una precisió de 6 decimals.
latitude No latitud corresponent amb una ubicació de la superfície terrestre en la que es mostrarà el marcador. Valor entre -90 i 90 amb una precisió de 6 decimals.
address No Si no coneixes la longitud i latitud, pots indicar l'adreça de la següent manera: carrer nº, ciutat, país
info No Informació que es mostrarà al fer click en el marcador. Permet introduir codi HTML

Exemple 1

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:gMap.

Pàgina JSP

Cal especificar un contenidor amb una mida específica, sinó ocuparà el 100% de la pantalla. 

<tr><td style="width: 800px; height: 600px;">
<fwk:gMap id="mapa1" address="Canigó" />
</td></tr>


Exemple 2

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:gMap i fwk:gMapMarker.

Pàgina JSP

Cal especificar un contenidor amb una mida específica, sinó ocuparà el 100% de la pantalla. 

<tr><td style="width: 800px; height: 600px;">
	<fwk:gMap id="mapa1" address="Canigó" zoom="15" showLargeMapControl="true" showScaleControl="true" showMapTypeControl="true" mapType="G_PHYSICAL_MAP" enableDragging="true" enableDoubleClickZoom="true" onclick="mostraCoordenades">
		<fwk:gMapMarker address="Canigó" info="<b>Canigó</b> the Framework <br /><img src=\"http://canigo.ctti.gencat.cat/confluence/download/resources/com.atlassian.confluence.themes.ctti:ctti/images/company_logo.gif\" /><br /><img src=\"http://canigo.ctti.gencat.cat/confluence/download/resources/com.atlassian.confluence.themes.ctti:ctti/images/OTCanigo_logo175x36.png\" />"></fwk:gMapMarker>
	</fwk:gMap>
</td>
</tr>
<tr><td style="width: 800px; height: 200px; background-color: blue;">
	<fwk:gMap id="mapa2" latitude="20" longitude="-2" zoom="5">
		<fwk:gMapMarker latitude="20.5" longitude="-1.8"></fwk:gMapMarker>
	</fwk:gMap>
</td>
</tr>


Javascript

function mostraCoordenades(latitud, longitud) {
	alert(latitud + ", " + longitud);
}
