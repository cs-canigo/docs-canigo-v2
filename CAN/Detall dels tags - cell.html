Detall dels tags - cell

Defineix una cel.la dins d'una quadrícula (grid) qu epermet insertar texte, amb format HTML. La cel.la pot estendre's a més d'una columna.

Estén el tag layout:cell ( http://struts.improve-technologies.com/doc/tags/layout.html#cell ) de Struts-Layout

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/cell.html

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:cell

L'exemple crea un grid dues columnes. Com que inclou quatre tags, es crearan automàticament 2 files. En mig d'aquestes dues files es defineix un cell que ocupa les dues columnes i conté un texte HTML.

Pàgina JSP

<%@ include file="/WEB-INF/jsp/includes/fwkTagLibs.jsp" %>

<fwk:form action="provesTagsLayout.do"
          styleId="myActionForm"
          key="jsp.provesTags.ProvesTagsSimple.form"
          reqCode="show"
          method="post"
          layout="true">
    <fwk:grid cols="2">
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp1"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp2"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>

        <fwk:cell colspan="2">
            <hr>un texto con <i>formato HTML</i> <u>insertado</u> en la tabla<hr>
        </fwk:cell>

        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp3"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="8"
                  size="8"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp4"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
    </fwk:grid>
</fwk:form>

Arxius d'internacionalització

Els elements de layout no son visibles i, per tant, no tenen textes asssociats.
