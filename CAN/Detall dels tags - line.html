Detall dels tags - line

Defineix una linia. Tots els tags que contingui es distribuiran horitzontalment, un al costat de l'altre.

És pràcticament igual que el tag row excepte que els elements de les diferentes linies queden alineats entre sí. L'efecte es pot observar comparant les imatges dels exemples d'aquest tag i del tag row.

Estén el tag layout:line ( http://struts.improve-technologies.com/doc/tags/layout.html#line ) de Struts-Layout

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/line.html

Atributs heredats

No hereda cap atributs de layout:line.

Atributs específics de Canigó

Atribut Funció
space (true | false) determina si s'inserta o no espai extra per tal d'alinear els camps de varies linies entre sí

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:line

L'exemple crea dues files que contenen tres camps de texte en cada una.

Cal observar que els elements dels diferents lines estàn alineats entre sí, a diferència del que s'aconsegueix utilitzant dos row.

Pàgina JSP

<%@ include file="/WEB-INF/jsp/includes/fwkTagLibs.jsp" %>

<fwk:form action="provesTagsLayout.do"
          styleId="myActionForm"
          key="jsp.provesTags.ProvesTagsSimple.form"
          reqCode="show"
          method="post"
          layout="true">
    <fwk:line>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp1"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp2"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp3"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="8"
                  size="8"
                  property="camp1"/>
    </fwk:line>
    <fwk:line>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp4"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>

        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp5"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp6"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="12"
                  size="12"
                  property="camp1"/>
    </fwk:line>
</fwk:form>

Arxius d'internacionalització

Els elements de layout no son visibles i, per tant, no tenen textes asssociats.
