Detall dels tags - label

Defineix una etiqueta (un texte constant).

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/label.html

Normalment el tag fwk:label no s'hauria d'utilitzar. Les etiquetes associades als diversos elements que composen els formularis es defineixen conjuntament amb l'element (atribut key) i amb l'atribut layout=true les parelles etiquetes-elements queden correctament alineades.

Atributs específics de Canigó

Atribut Funció
id

services  
styleClass  
styleId identificador del tag, per lligar-lo amb la configuració injectada amb Spring

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:label

Pàgina JSP

<%@ include file="/WEB-INF/jsp/includes/fwkTagLibs.jsp" %>

<fwk:form action="provesTagsSimple.do"
          styleId="myActionForm"
          key="jsp.provesTags.ProvesTagsSimple.form"
          reqCode="show"
          method="post"
          layout="true">
    (...)
    <fwk:label styleId="lbl1"
               key="jsp.provesTags.ProvesTagsSimple.lbl1"
               id="camp7"/>
    (...)
</fwk:form>

Arxius d'internacionalització

jsp.provesTags.ProvesTagsSimple.lbl1=Una etiqueta

Bugs

Les etiquetes definides amb aquest tag no queden correctament alineades amb la resta d'etiquetes definides de forma conjunta amb un altre element.
