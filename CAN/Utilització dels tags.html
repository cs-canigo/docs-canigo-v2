Utilització dels tags

Els formularis construits a partir del Servei de Tags de Canigó estàn composats per Camps i distribuits segons un Layout.

A continuació es mostra un exemple d'un formulari que inclou els diferents tipus de Camps disponibles. Els tags visibles es complementen amb un conjunt de tags de layout que permeten controlar la distibució dels elements visibles en el formulari i un conjunt de tags de validacions. A la taula de sota es pot accedir a la documentació concreta de cada Tag.
