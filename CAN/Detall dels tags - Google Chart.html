Detall dels tags - Google Chart

Aquests tags integren l'api de Google Chart amb Canigó.

Per a utilitzar els mapes de google s'han creat dos tags:
# fwk:gChart: crea un gràfic del tipus desitjat.

Aquests tags no implementen totes les funcionalitats de Google Chart, podeu trobar més informació de l'api de Google a ( [http://code.google.com/intl/es-ES/apis/charttools/] ).

fwk:gChart

Mostra una imatge d'un gràfic generat a partir de les dades especificades.

Atributs

Atribut Obligatori Funció
type Sí Tipus de gràfic:
- Horitzontal: GoogleChartTag.TYPE_BAR_HORIZONTAL_STACKED
- Horitzontal Agrupat: GoogleChartTag.TYPE_BAR_HORIZONTAL_GROUPED
- Vertical: GoogleChartTag.TYPE_BAR_VERTICAL_STACKED
- Vertical Agrupat: GoogleChartTag.TYPE_BAR_VERTICAL_GROUPED
- Circular: GoogleChartTag.TYPE_PIE
- Circular 3D: GoogleChartTag.TYPE_PIE_3D
- Línies: GoogleChartTag.TYPE_LINE
- Línies Simple: GoogleChartTag.TYPE_LINE_SIMPLE
- Punts: GoogleChartTag.TYPE_LINE_POINTS
width Sí Amplada del gràfic. Valor màxim de l'amplada per l'alçada no pot superar els 300.000 píxels
height Sí Alçada del gràfic. Valor màxim de l'amplada per l'alçada no pot superar els 300.000 píxels
color No Color del gràfic. Valors en hexagesimal separats per comes
legend No Llegenda del gràfic. Valor possibles:
- no: No es mostra llegenda
- default: Es mostra la llegenda per defecte
- qualsevol altre valor es concatenarà a chdl, que es el que utilitza l'api de google per a mostrar la llegenda.
axisLabels No Etiquetes del gràfic. Valor possibles:
- no: No es mostra cap etiqueta
- default: Es mostren les etiquetes per defecte
- qualsevol altre valor es concatenarà a chxt, que es el que utilitza l'api de google per a mostrar les etiquetes en els eixos.
data No Dades que s'utilitzen per a mostrar el gràfic. Aquestes dades, s'informaran en un LinkedHashMap i aquest podrà contenir altres LinkedHashMap
customChart No Ofereix la possibilitat d'introduïr directament una cadena tal i com s'indica en l'api de google per a mostrar un gràfic. Si s'informa aquest atribut, tots els altres valors seran ignorats.
customOptions No Ofereix la possibilitat d'introduïr directament una cadena d'opcions addicionals tal i com s'indica en l'api de google.
