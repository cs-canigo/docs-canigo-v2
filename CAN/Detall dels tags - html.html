Detall dels tags - html

Inserta el següent codi html:

<html>
  <head>
    <base href="http://localhost/canigo-env/">
    <link rel="stylesheet" href='/canigo-env/css/default.css' type="text/css">
    <script language="Javascript" src="/canigo-env/struts/javascript.js"></script>
    <script>
      var imgsrc="/canigo-env/images/";
      var scriptsrc="/canigo-env/struts/";
      var langue="ca";
      var contextPath="/canigo-env";
    </script>
  </head>
  <body>
    <table cellspacing="0" cellpadding="10" width="100%" border="0">
      <tr>
        <td>
(...)
        </td>
      </tr>
    </table>
  </body>
</html>

Com que normalment els formularis estàn integrats en el contexte d'una pàgina amb molts altres elements (capçalera, menus, etc.), la inicialització de la pàgina es fà fora del formulari i no és responsabilitat d'aquest. Per tant, en aquest contexte, no cal utilitzar aquest tag.

Per insertar codi html arbitrari en un formulari pot ser útil el tag fwk:cell.

Estén el tag layout:html ( http://struts.application-servers.com/doc/index.html#html ) de Struts-Layout.

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/html.html
