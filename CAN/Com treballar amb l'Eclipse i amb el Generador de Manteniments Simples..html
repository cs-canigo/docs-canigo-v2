Pasos a seguir per treballar amb el Generador de Manteniments Simples i Eclipse.

El Generador de Manteniments Simples és una eina que permet generar manteniments simples a partir de les metadates d'una taula d'una BD determinada. El Generador de Manteniments Simples es pot incorporar directament en un projecte d'Eclipse. D'aquesta manera és molt més fàcil veure els resultats del generador (codi font generat) així com la incorporació dels fitxers generats en una estructura de directoris determinada.

Lectura prèvia recomanda

Es recomana, abans de continuar amb els pasos descrits a continuació, llegir l'apartat Generador de manteniments simples.


Els pasos a seguir per incorporar el Generador de Manteniments Simples en Eclipse són els següents:

Pas 1. Descarregar-se el Generador de manteniments simples (Generador de manteniments simples): llibreries, .jar i build.xml.

Pas 2. Incorporar el generador de manteniments a l'Eclipse. Hi ha dues maneres de fer-ho:

  • Generar un nou projecte a l'Eclipse: File > New Project... > Java Project i copiar els fitxers anteriors dintre d'aquest projecte nou.
  • Copiar les llibreries del .zip al directori de llibreries del projecte, copiar el .jar i el build.xml al projecte.

Pas 3. Generar el fitxer generator.properties:

#En les dades de connexió amb el SGBD si database.sgbd=M (s'estableix  connexió amb MySQL), si és database.sgbd=O (s'estableix connexió amb  Oracle).

database.host=bna-s007.es.int.atosorigin.com
database.port=1521
database.username=canigo
database.password=canigo
database.schema=canigo
database.service=Legolas
database.sgbd=O

directories.workingDirectory=C:/CanigoGeneratorTest
project.projectName=canigo-samples-jPetstore
project.rootDir=net/gencat/ctti/canigo/samples/jpetstore
project.rootPackage=net.gencat.ctti.canigo.samples.jpetstore
project.extendLayout=site.altLayoutPetStore
project.menuJSP=/src/main/webapp/WEB-INF/jsp/includes/left.jsp

#Si no s'informa crudModule es crearà un mòdul amb el nom de la taula de la que es genera el manteniment
# simple. Si s'informa crudModule es crearà un mòdul en el que s'afegirà tot el codi generat pel manteniment simple.
i18n.menuDir=/src/main/resources/i18n
database.convertTable=PERMIS_APLICACIO
project.crudModule=permis_aplicacio
project.attributeFK=id_aplicacio
project.sequenceName=SEQ_PERMIS_APLICACIO

Per més informació llegir l'apartat 7 de la documentació referent al Generador de manteniments simples.

Pas 4. Incorporar el build.xml a la vista Ant de l'Eclipse: Window > Show View > Ant

Pas 5. Arrastrar el fitxer build.xml a la vista Ant.

Pas 6. Fer doble click sobre la tasca Ant canigoGenerator per tal de generar el manteniment simple desitjat.
