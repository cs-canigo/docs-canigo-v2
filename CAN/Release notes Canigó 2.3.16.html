Canigó - Release notes Canigó 2.3.16

Release notes de la versió 2.3.16 de Canigó

Bugs fixats a Canigó 2.3.16:

Canvis
  • Canvi CAN-1413: Integració GICAR amb Filenet
    Context: Donar capacitat al servei de seguretat per recuperar el token de sessió de GICAR amb l'objectiu d'utilitzar-lo per poder fer autenticacions sense que es demani constantment usuari/password.
    Servei afectat: canigo-services-security
    Solució: Modificació del servei per incorporar la nova funcionalitat.

  • Canvi CAN-1407: Configurar la orintació de pàgina en l'exportació de llistats a PDF
    Context: Donar capacitat a les aplicacions per configurar l'orientació del PDF generat amb el servei d'exportació de dades a PDF.
    Servei afectat: canigo-services-web
    Solució: Modificació del servei per incorporar la nova funcionalitat
