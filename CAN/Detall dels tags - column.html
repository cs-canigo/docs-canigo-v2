Detall dels tags - column

Defineix una columna. Tots els tags que contingui es distribuiran verticalment, un sobre l'altre.

Estén el tag layout:column ( http://struts.improve-technologies.com/doc/tags/layout.html#column ) de Struts-Layout

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/column.html

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:column

L'exemple crea una columna amb sis camps de texte.

Pàgina JSP

<%@ include file="/WEB-INF/jsp/includes/fwkTagLibs.jsp" %>

<fwk:form action="provesTagsLayout.do"
          styleId="myActionForm"
          key="jsp.provesTags.ProvesTagsSimple.form"
          reqCode="show"
          method="post"
          layout="true">
    <fwk:column>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp1"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp2"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp3"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="8"
                  size="8"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp4"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="15"
                  size="15"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp5"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="10"
                  size="10"
                  property="camp1"/>
        <fwk:text key="jsp.provesTags.ProvesTagsSimple.camp6"
                  styleId="camp1"
                  mode="E,E,E"
                  layout="true"
                  maxlength="12"
                  size="12"
                  property="camp1"/>
    </fwk:column>
</fwk:form>

Arxius d'internacionalització

Els elements de layout no son visibles i, per tant, no tenen textes asssociats.
