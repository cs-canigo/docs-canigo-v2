Detall dels tags - radio

Defineix un botó de radio.

Normalment els botons de radio s'utilitzaran en grups de 2 o més, lligats donant el mateix valor a l'atribut property, per tal de seleccionar els diferents valors que es poden assignar.

Estén el tag layout:radio ( http://struts.application-servers.com/doc/tags/select.html#radio ) de Struts-Layout.

La documentació auto-generada del tag és a http://canigo.ctti.gencat.net/confluence/canigodocs/site/canigo2_2/tlddoc/fwk/radio.html

Atributs heredats

Els atributs heredats de layout:radio son: accesskey, arg0, arg1, arg2, arg3, arg4, bundle, hint, key, layoutId, maxlength, name, onblur, onchange, onclick, ondblclick, onfocus, onkeydown, onkeypress, onkeyup, onmousedown, onmousemove, onmouseout, onmouseover, onmouseup, onselect, policy, property, size, styleClass, tabindex, tooltip, value.

Atributs específics de Canigó

Atribut Funció
layout (true / false) Especifica si s'ha de generar o no etiqueta associada
services  
styleId Identificador del tag, per lligar-lo amb la configuració injectada amb Spring.
tooltipKey Clau del literal del missatge a mostrar dins el tooltip.
tooltipOptions  
tooltipTitleKey Clau del literal a mostrar si volem mostrar un títol com a capçalera del tooltip

Per utilitzar els Tooltips cal incloure un conjunt d'arxius JavaScript i CSS:

  • el full d'estil tooltips.css
  • els arxius de scripts domLib.js i domTT.js
<LINK REL=StyleSheet
      HREF="css/tooltips/tooltips.css"
      TYPE="text/css"
      MEDIA=screen>

<script type="text/javascript"
        src="<c:url value="/scripts/tooltips/dom_tooltip/domLib.js"/>">
</script>
<script type="text/javascript"
        src="<c:url value="/scripts/tooltips/dom_tooltip/domTT.js"/>">
</script>

Exemple

A continuació es presenta un fragment de pàgina que utilitza el tag fwk:radio

Es defineixen tres botons, que permeten seleccionar entre tres valors diferents (valor_1, valor_2, valor_3) per la propietat comuna camp4.

Pàgina JSP

<script type="text/javascript"
        src="<c:url value="/scripts/tooltips/dom_tooltip/domLib.js"/>">
</script>
<script type="text/javascript"
        src="<c:url value="/scripts/tooltips/dom_tooltip/domTT.js"/>">
</script>
<LINK REL=StyleSheet
      HREF="css/tooltips/tooltips.css"
      TYPE="text/css"
      MEDIA=screen>

<fwk:form action="provesTagsSimple.do"
          styleId="myActionForm"
          reqCode="show"
          method="post"
          layout="true">
    (...)
    <fwk:radio key="jsp.provesTags.ProvesTagsSimple.camp4.1"
               styleId="camp4.1"
               property="camp4"
               tooltipTitleKey="jsp.provesTags.ProvesTagsSimple.camp4.tipTitle"
               tooltipKey="jsp.provesTags.ProvesTagsSimple.camp4.1.tip"
               value="valor_1"/>

    <fwk:radio key="jsp.provesTags.ProvesTagsSimple.camp4.2"
               styleId="camp4.2"
               property="camp4"
               tooltipTitleKey="jsp.provesTags.ProvesTagsSimple.camp4.tipTitle"
               tooltipKey="jsp.provesTags.ProvesTagsSimple.camp4.2.tip"
               value="valor_2"/>

    <fwk:radio key="jsp.provesTags.ProvesTagsSimple.camp4.3"
               styleId="camp4.3"
               property="camp4"
               tooltipTitleKey="jsp.provesTags.ProvesTagsSimple.camp4.tipTitle"
               tooltipKey="jsp.provesTags.ProvesTagsSimple.camp4.3.tip"
               value="valor_3"/>
    (...)
</fwk:form>
