Canigó - Release notes Canigó 2.3.13

Release notes de la versió 2.3.13 de Canigó

Peticions incorporades a Canigó 2.3.13:

Canvis
  • Canvi CAN-1234: Connector funcional PICA - SARCAT
    Context: Connector funcional PICA - SARCAT
    Connector afectat: canigo-connectors-pica
    Solució: Implementació de la funcionalitat pel connector
  • Canvi CAN-1283: Enviar correu amb inlines
    Context: Enviar correu amb inlines bassat en plantilles html
    Connector afectat: canigo-service-mail
    Solució: Modificació del servei de mail per fer que es pogui enviar mails amb plantilles html que continguin imatges inline.
  • Canvi CAN-1294: Consulta sobre classe net.gencat.ctti.canigo.services.mail.impl.SpringMailServiceImpl
    Context: Encriptar usuari/password dels arxius de properties del servei de mail
    Connector afectat: canigo-service-configuration
    Solució: Modificació del servei per tal de que es pogui codificar qualsevol dada d'un arxiu de properties de qualsevol servei del framework
Incidències
  • Incidència CAN-1311: Càrrega log4j per entorns en Weblogic-Windows
    Context: El servei de traces no detecta correctament el path de l'arxiu de properties quan aquest era carregat des d'un WL sobre un SO windows
    Servei afectat: canigo-service-logging
    Solució: Modificació del servei de traces per solventar el problema
